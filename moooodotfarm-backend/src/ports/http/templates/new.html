how to get your cow.txt listed

If you want your cow added to the herd, email the URL of your cow.txt to boreq@0x46.net.

As some cows can be shy and other cows can be brave please clearly state whether the cow URL should be VISIBLE or CENSORED, and make sure the URL you send is NOT censored.

Cows can change their minds at any time, so feel free to email whenever you want to switch whether the cow’s name is visible or censored.

The website needs the uncensored URL to verify that cow.txt is present, but if you request the URL to be censored, the uncensored URL will never be made publicly accessible via any means.
